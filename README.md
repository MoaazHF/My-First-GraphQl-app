# The Grind Hotel — Hotel Management API

Backend API built with NestJS, GraphQL (Apollo), TypeORM, 
PostgreSQL, Redis, and RabbitMQ.

## Features
- Room & booking management with GraphQL queries/mutations
- DataLoader for N+1 query optimization  
- Redis caching layer
- Docker Compose setup

## Stack
NestJS · GraphQL · TypeORM · PostgreSQL · Redis · RabbitMQ · Docker
